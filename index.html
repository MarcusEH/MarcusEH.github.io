Actualize Capstone: Turnout

An application that encourages planning and attendance of events with friends. The app allows users to create group events, have the app select the time and type of event based on user preferences and displays not only their Turnout events but also the individual user’s recent Google Calendar events. It also allows interaction with other users via email, group invites and comments. The application was made with a Rails backend and a Vue.js frontend framework with Node.js. It uses Figaro, ActiveMailer on the backend and Axios, Google API, Google OAuth, Full-Calendar as well as Moment.js on the frontend.
